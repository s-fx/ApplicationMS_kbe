## Projektübersicht <br>
Das Projekt ist das Backend für eine E-Commerce Plattform, auf der Nutzer Laptops kaufen und verkaufen können. Sie setzt sich zusammen aus vier Microservices(Repos):
<br>

- ApiGateway_kbe
- DatawarehouseMS_kbe
- CalculatorMS_kbe
- ApplicationMS_kbe
